# Mithila-Lipi

**Mithila-Lipi** is a free, browser-based tool to convert **Maithili text written in Devanagari** into the traditional **Tirhuta (मिथिलाक्षर)** script.

The project is built to support learning, writing, and preservation of the Maithili language and its original writing system.

🌐 **Live website**  
https://mithila-lipi.netlify.app/

---

## ✨ Features

- Translate **English → Hindi (Devanagari)** inside the app
- Convert Maithili text from **Devanagari → Tirhuta**
- One-click **English → Hindi → Tirhuta** pipeline
- Fully browser-based (no backend, no tracking)
- Unicode-correct Tirhuta output
- Mobile and desktop friendly
- One-click copy to clipboard
- Clean UI inspired by Mithila art

---

## 🧭 How to use

### Step 1 — English → Hindi → Tirhuta (one click)
1. Enter English text in the **English** box  
2. Click **Translate + Convert**  
3. Copy the Tirhuta output

### Step 2 — Optional direct Devanagari → Tirhuta
1. Paste Hindi/Maithili (Devanagari) text into the second input box  
2. Click **Convert**  
3. Copy the Tirhuta output

---

## 🧠 About Tirhuta (मिथिलाक्षर)

Tirhuta is the traditional script of the Maithili language, historically used across the Mithila region.
While modern Maithili is often written in Devanagari, Tirhuta remains an important part of Maithili’s
cultural and literary heritage.

Mithila-Lipi does not translate meaning — it only converts the **writing system**.
